# POW-Capstone2025
Repository for POW Capstone Project Fall 2025 UPRM
Needed PIP installs
- torch
- transformers
- pandas
- kagglehub
- bs4


Preferably install them inside your Python Venv (Optional)


How to Run:
Install dependencies in requirements.txt 
Run function train_model() inside of DistilBERT.py file
Once model has been fine tuned, run server.py file using this command: uvicorn server:app --reload --host 0.0.0.0 --port 8000
Now you can install the extension inside chrome and use it in an outlook email.
